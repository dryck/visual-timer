# ⏱️ Visual Timer

A beautiful, easy-to-see countdown timer designed for classrooms.

## 🌐 Live Demo
https://dryck.github.io/visual-timer

## ✨ Features

- 🎨 **Visual Progress Bar** - Changes color as time runs out (green → yellow → red)
- 🐢 **Animated Mascot** - Character that reacts to time passing
- 🔊 **Sound Alerts** - Audio notifications when time is up
- 📺 **Fullscreen Mode** - Visible from anywhere in the classroom
- ⚡ **Quick Presets** - One-click buttons for common times
- ⏸️ **Pause/Resume** - Flexible control during lessons

## 🎯 Perfect For

- ⏱️ **Timed activities** - Keep students focused and on track
- 🔄 **Transitions** - Smooth between lesson segments
- 📝 **Tests and quizzes** - Fair time management
- 🧘 **Mindfulness breaks** - Calm down periods
- 🎮 **Game time** - Fair turns and rotations

## 📖 How to Use

### Quick Start (Presets)
Click any preset button:
- 5 min - Quick discussion
- 10 min - Short activity
- 15 min - Standard exercise
- 20 min - Deep work
- 30 min - Major task
- 45 min - Full period

### Custom Time
1. Enter minutes and seconds
2. Click "Set"
3. Click "▶️ Start"

### During Timer
- **⏸️ Pause** - Freeze the timer (for interruptions)
- **🔄 Reset** - Start over
- **⛶ Fullscreen** - Make it classroom-visible

### Visual Cues
| Time Left | Color | Meaning |
|-----------|-------|---------|
| >70% | 🟢 Green | Plenty of time |
| 30-70% | 🟡 Yellow | Keep working |
| <30% | 🔴 Red | Hurry up! |
| Done | 🎉 Flash | Time's up! |

## 💡 Classroom Tips

**For Young Students:**
- Use the mascot as a visual anchor
- "When the turtle stops, we stop"
- Color changes help non-readers

**For Time Management:**
- Set 2-minute warning for transitions
- Use for "think time" before answering
- Pomodoro technique (25 min work, 5 min break)

**For Engagement:**
- "Beat the timer" challenges
- Countdown to exciting activities
- Fair sharing time

**Fullscreen Tips:**
- Project on board for whole class
- Use on tablet for small groups
- Phone works for individual work

## 🔊 Sound Options

Toggle sound on/off with the checkbox.

**Sounds include:**
- Start beep
- Warning tones
- Alarm when finished
